java編譯與執行

1. 用編輯器編寫HelloWorld.java
2. javac編譯
3. java執行

HelloWorld.java
public class HelloWorld {
  public static void main(String[] argv) {
    System.out.println("Hello World\n");
   }
}

javac編譯
javac HelloWorld.java

java執行
java HelloWorld

編譯多個檔案

1. 用編輯器編寫2個檔案,分別為Hello.java及Main.java
2. javac編譯
3. java執行

Hello.java
public class Hello {
   public void sayHello() {
     System.out.println("Hello\n");
   }
}

Main.java
public class Main {
   public static void main(String[] argv) {
     Hello hello = new Hello();
     hello.sayHello();
   }
}

編譯
javac *.java

執行
java Main
